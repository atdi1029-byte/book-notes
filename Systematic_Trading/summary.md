# Systematic Trading — Robert Carver (2015)

A former hedge fund portfolio manager's complete framework for building
trading systems that remove emotion from financial decision-making.
Carver organizes around three trader types — asset allocating investors
using ETFs with no forecasts, semi-automatic traders combining
discretionary views with systematic risk management, and staunch
systems traders running fully automated rule-based strategies. The book
provides a modular framework where trading rules produce scaled
forecasts (expected absolute value of 10, capped at ±20), which flow
through volatility targeting, position sizing, and portfolio
construction. Key principles include avoiding over-fitting (you need
30+ years to distinguish most rules), using handcrafting instead of
unstable Markowitz optimization for portfolio weights, setting risk
via Half-Kelly criterion, and never spending more than one-third of
expected Sharpe ratio on trading costs. Carver's own system trades
40+ futures with eight rules across five themes, achieving a
back-tested Sharpe ratio of 1.0 — which he runs at just 37%
volatility target after applying all his pessimism adjustments.
